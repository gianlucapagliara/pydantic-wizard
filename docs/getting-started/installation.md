# Installation

## Using pip

```bash
pip install pydantic-wizard
```

## Using uv

```bash
uv add pydantic-wizard
```

## Optional: Web UI

To include the Streamlit-based web interface:

```bash
pip install pydantic-wizard[web]
```

Or with uv:

```bash
uv add pydantic-wizard --extra web
```

## Requirements

- Python 3.13+
- Pydantic v2
