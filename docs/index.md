# Pydantic Wizard

Interactive wizard-style CLI for configuring Pydantic v2 models, with YAML serialization.

## Features

- **Interactive prompts** for all Pydantic field types
- **YAML serialization** with metadata headers
- **Validation** against Pydantic models
- **Extensible** type handler registry
- **Rich terminal output** with syntax highlighting

## Quick Start

```bash
pip install pydantic-wizard
```

```bash
pydantic-wizard new myapp.config.AppConfig
```

See the [Getting Started](getting-started/installation.md) guide for more details.
